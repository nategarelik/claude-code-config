# /debug

## Purpose
Apply systematic debugging process to identify and fix issues.

## Process
1. **Observe** - Gather evidence about the bug
2. **Hypothesize** - Form theories about root cause
3. **Test** - Verify hypotheses with targeted investigation
4. **Fix** - Implement minimal solution
5. **Verify** - Confirm resolution

## Output
- Root cause identification
- Fix implementation
- Verification evidence
