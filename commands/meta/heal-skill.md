# /heal-skill

## Purpose
Fix a broken or underperforming skill by analyzing failures and improving.

## Process
1. **Identify symptoms** - What's not working?
2. **Review skill definition** - Check activation criteria, instructions
3. **Analyze usage patterns** - How is it being triggered?
4. **Propose improvements** - What changes would help?
5. **Implement and test** - Apply fixes, verify improvement

## Output
Updated skill file with improvements documented.
